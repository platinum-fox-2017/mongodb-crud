To Use Book API

| Route           | HTTP      | Description                                                |
| ----------------|:---------:| ----------------------------------------------------------:|
| /api/book       | GET       | Get all books                                              |
| /api/book       | POST      | Insert new book                                            |
| /api/book       | PUT       | Update a book                                              |
| /api/book       | DELETE    | Delete a book                                              |
